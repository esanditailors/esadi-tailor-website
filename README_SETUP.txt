ESADI TAILOR - MIRIGAMA Website
================================

This is a modern multi-page static website made with HTML, CSS and JavaScript.

Pages:
- index.html
- about.html
- services.html
- gallery.html
- booking.html
- contact.html

How to open:
1. Extract the ZIP file.
2. Open index.html in your browser.

How booking works:
- The booking form does not need a backend.
- When the customer submits the form, WhatsApp opens with the booking message.
- WhatsApp number is linked to: 077 600 1972

How to edit WhatsApp / Facebook / login link:
Open this file:
assets/js/main.js

Change these values:
whatsapp: "94776001972"
facebook: "https://www.facebook.com/share/1CpkiTvVPU/?mibextid=wwXIfr"
loginUrl: "https://Aruna1991.pythonanywhere.com"

Important:
- I used a placeholder Facebook link because the exact Facebook page URL was not provided.
- Replace it in assets/js/main.js after you get the exact Facebook page link.
